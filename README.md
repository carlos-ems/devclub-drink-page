# DevClub Drink Web Page
## (PT-BR)
Este projeto consiste na implementação de uma página web simples sobre drinks, desenvolvida como parte de uma ideia proposta no curso da DevClub (link: https://www.youtube.com/watch?v=QLbrpjfltFs&list=PLsFVybaG4mOBmLZH65Z5HbR0FieLJJ1_r).
A aplicação foi construída utilizando apenas HTML, CSS e JavaScript, com o objetivo de praticar conceitos fundamentais do desenvolvimento web, como estruturação de páginas, estilização visual e interatividade no lado do cliente.

O foco do projeto é aplicar boas práticas básicas de front-end, mantendo um código simples, organizado e de fácil compreensão.

## (EN-US)
This project consists of the implementation of a simple drink-themed web page, developed based on an idea proposed in a DevClub course (link: https://www.youtube.com/watch?v=QLbrpjfltFs&list=PLsFVybaG4mOBmLZH65Z5HbR0FieLJJ1_r).
The application was built using only HTML, CSS, and JavaScript, aiming to practice fundamental web development concepts such as page structure, visual styling, and client-side interactivity.

The main goal of this project is to apply basic front-end best practices while keeping the code simple, clean, and easy to understand.
